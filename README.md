# 🚀 HUOKAING THARA | Future ISP

A futuristic web interface concept for a next-generation satellite-based ISP system.
This project simulates a cyberpunk-style network dashboard with security layers, holographic UI, and dynamic IP coordination.

---

## 🧠 Overview

**HUOKAING THARA** is a front-end concept that represents a fictional satellite ISP control interface.
It combines modern web technologies with a sci-fi aesthetic to create an immersive user experience.

---

## ✨ Features

* 🔐 **Neural Handshake Security Layer**

  * Google reCAPTCHA integration
  * Simulates authentication before accessing the system

* 🛰️ **Satellite Interface Dashboard**

  * Glitch-style animated header
  * Real-time styled system status display

* 📡 **Dynamic IP Coordination Panel**

  * Simulated IP tracking / assignment system
  * Futuristic “bank panel” visualization

* 🎨 **Cyberpunk UI Design**

  * Neon color palette
  * Hologram-inspired layout
  * Animated effects (CSS + JavaScript)

## 🔑 reCAPTCHA Setup

This project uses **Google reCAPTCHA**.

To configure:

1. Go to: [https://www.google.com/recaptcha/admin](https://www.google.com/recaptcha/admin)
2. Create a new site key
3. Replace your key inside the script configuration
